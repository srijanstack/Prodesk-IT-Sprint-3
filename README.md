# Prodesk IT Sprint 3

This is a part of Prodesk IT internship

## Features

- Search for any GitHub user by username.
- View profile details including avatar, bio, followers, following, and public repositories.
- Explore the latest repositories with stars, forks, and primary programming language.
- Compare two GitHub users in Battle Mode based on total repository stars.
- Automatically detect winners, losers, and ties.
